# v1.0.10 - optional Windows code signing

Date: 2026-06-21

## 中文

### 目标

为 Windows 构建增加可选代码签名通道，降低 Application Control / 企业策略环境下新构建 `shiyi.exe` 被拦截后无法继续稳定测速的问题。

### 修改

- `scripts/build-windows.ps1` 增加 `-SignThumbprint`、`-SignPfxPath`、`-SignPfxPassword` 和 `-TimestampUrl` 参数。
- 构建脚本会在打包前对 `shiyi.exe` 和 `start.exe` 签名；未配置证书时继续无签名构建。
- GitHub Release workflow 支持从 `WINDOWS_SIGNING_PFX_BASE64` 和 `WINDOWS_SIGNING_PFX_PASSWORD` secrets 导入 PFX 证书。
- README 和包内 README 补充签名构建说明。

### 验证

- 无证书构建命令：`powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\build-windows.ps1 -Version v1.0.10-test`
- 构建退出码 0，日志包含 `Code signing skipped: no certificate configured.`
- `Get-AuthenticodeSignature dist\shiyi.exe,dist\start.exe` 显示本地测试构建为 `NotSigned`。
- `shiyi.exe --help` 正常运行。
- 测试后无 `shiyi`、旧 `profanity*`、`TronStudio` 或 `start` 残留进程。

### 性能

RTX 3070 当前样本：

- 随机单目标 3 秒：351.127 MH/s
- 全 0 初始私钥、后 16 位向上、后 8 位地址测试 3 秒：333.413 MH/s

400 MH/s 目标尚未达成。本版本不声称性能目标完成，只为后续稳定运行和测速补齐签名发布能力。

## English

### Goal

Add an optional Windows code-signing path so newly built `shiyi.exe` can be run and benchmarked more reliably under Application Control or enterprise policy.

### Changes

- Added `-SignThumbprint`, `-SignPfxPath`, `-SignPfxPassword`, and `-TimestampUrl` to `scripts/build-windows.ps1`.
- The build script signs `shiyi.exe` and `start.exe` before packaging when a certificate is configured; unsigned builds still work when no certificate is configured.
- The GitHub Release workflow can import a PFX certificate from `WINDOWS_SIGNING_PFX_BASE64` and `WINDOWS_SIGNING_PFX_PASSWORD` secrets.
- README and package README document the signing build path.

### Verification

- No-certificate build command: `powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\build-windows.ps1 -Version v1.0.10-test`
- Build exit code 0; log contains `Code signing skipped: no certificate configured.`
- `Get-AuthenticodeSignature dist\shiyi.exe,dist\start.exe` reports the local test build as `NotSigned`.
- `shiyi.exe --help` runs successfully.
- No leftover `shiyi`, old `profanity*`, `TronStudio`, or `start` processes after tests.

### Performance

Current RTX 3070 samples:

- Random single-target, 3 seconds: 351.127 MH/s
- All-zero initial key, last-16 upward, address last-8 test, 3 seconds: 333.413 MH/s

The 400 MH/s target has not been reached. This version does not claim performance completion; it adds signing release support for more stable future runs and benchmarks.
