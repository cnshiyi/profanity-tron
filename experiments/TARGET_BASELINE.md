# 外部对标基线

这个文件记录外部样本的实测速率，用于衡量本项目当前与目标之间的差距。

## 对标样本

- 文件: `gd_gpu.exe`
- 场景: `生成8A+地址.bat`
- 模式: `lianhao`
- 参数: `--lianhao yes --hou 8`
- GPU: `NVIDIA GeForce RTX 3070`

## 实测结果

在重启电脑后的复跑中，程序已经成功进入真实计算流程，且测试结束后已确认完全退出。

- 冷启动爬升: `61.021 -> 155.848 -> 187.457 -> 202.056 -> 211.780 MH/s`
- 稳定区间: `248 ~ 250 MH/s`
- 抓到的峰值: `250.193 MH/s`

## 当前意义

它是当前最明确的追赶目标。

以本仓库最近一次主工程专用快路径版本为例：

- 本项目: `120.565 MH/s`
- 外部对标: `250.193 MH/s`
- 当前差距: 约 `2.07x`

## 后续要求

后续每一轮性能优化，除了和历史版本对比，也要和这个对标速度对照：

1. 是否缩小了与 `250.193 MH/s` 的差距
2. 哪条路径最可能逼近或超过该水平
3. 是否需要继续拆分模式专用 kernel、引入两段式评分或 binary cache
